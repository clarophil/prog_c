Etape 1 :  Installer Visual Code

Visual Code est un éditeur de code développé par Microsoft pour Windows, Linux et macOS.  Il permet le développement dans de nombreux langages ; python, java, C, ...

Cet éditeur est open source et gratuit d'utilisation à ne pas confondre avec Visual Studio qui est plus complexe d'utilisation.

Vous pouvez le télécharger sur cette page https://code.visualstudio.com/download

Etape 2: Installer le compilateur et le debugger

Il existe de nombreux compilateurs et de nombreuses installations possibles, en voici une :

  1. Ouvrez la page https://code.visualstudio.com/docs/languages/cpp
  2. Sélectionner un compilateur en fonction de votre système d'exploitation
    1. Pour Windows https://sourceforge.net/projects/mingw-w64/files/Toolchains%20targetting%20Win32/Personal%20Builds/mingw-builds/installer/mingw-w64-install.exe/download
  3. Télécharger et installer le compilateur
    1. Lancer  l'installation en tant qu'administrateur
    2. Quand  tous les paquets sont installés, fermer la fenêtre
  4. Pour vérifier que tout est en ordre, ouvrez une fenêtre de commande
    1. tapez gcc --version
    2. taper gdb --version

Etape 3 : Configurer le compilateur pour Windows

  1. Rechercher le chemin du compilateur dans votre explorateur et copier le chemin de la barre d'adresse
  2. Dans la barre de recherche de Windows, taper environnement et sélectionner Modifier les variables d'environnement système
  3. Cliquer sur le bouton Variables d'environnement dans le base de la fenêtre
  4. Cliquer sur Path puis le bouton Modifier
  5. Vérifier que e chemin du compilateur est présent sinon cliquer sur Nouveau et coller le chemin
  6. Cliquer sur Ok dans toutes les fenêtre pour valider

Etape 4 : Installer l'extension Microsoft C/C++ extension

  1. Ouvrir Visual Code
  2. Cliquer sur l'icône Extensions dans la barre de menu à gauche
  3. Recherche c++
  4. Cliquer sur Installer

Etape 4 : Associer le compilateur à Visual Code

  1. Ouvez Visual Code
  2. Dans le menu Terminal, cliquer sur Configure Default Build Task
    1. Sélectionner C/C++:gcc.exe build active file -> le fichier tasks.json est créé avec la bonne configuration

Etape 5 : Compiler votre script

  1. Dans le menu Terminal, cliquer sur Run task
  2. Sélectionner C/C++:gcc.exe build active file
    1. Une fenêtre de commande s'ouvre et un message de compilation s'affiche
    2. L'exécutable apparaît sur la gauche dans l'explorateur

Etape 6 : Lancer l'exécutable
